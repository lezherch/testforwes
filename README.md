# testforwes

You need to download Jmeter test plan

1.- Download JMeter from its website  https://apache.inspire.net.nz//jmeter/binaries/apache-jmeter-5.4.1.zip  <br/>
2.- JMeter script is here  -> Westpac_Jmeter_TestPlan_Jul2021.jmx <br/>
3.- Data required lambomodelsparams.csv <br/>

Data file  needs to be loaded into  data folder <br/>
I.e   apache-jmeter-5.4.1/data/lambomodelsparams.csv<br/>

4.- To Execute Jmeter Test plan follow the next steps: <br/>
4.1 You need to open it with Jmeter (Go to apache-jmeter-5.4.1\bin\jmeter.bat) <br/>
4.2 open Script  Westpac_Jmeter_TestPlan_Jul2021.jmx<br/>
(If you are asked to update or download any additonal component please do)<br/>
4.3  Press start button  (Green arrow)<br/>
